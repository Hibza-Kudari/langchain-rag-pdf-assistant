# PDF RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using:

* Python
* FAISS
* Sentence Transformers
* Ollama
* Llama 3.2
* Streamlit

## Features

* Chat with PDF documents
* Semantic search using vector embeddings
* Local LLM inference using Ollama
* Streamlit web interface
* Chat history
* Source citations
